# TrebleApp Japanese
Magiskモジュールで日本語をサポートしたPhh Treble Settingsをインストールします。

「Phh Treble Settings」が「Phh Treble 設定」に変っていれば日本語での設定が可能になります。

## 既知の問題
一部の項目が動作しない場合があります。

一部のGSIでテキストの表示がおかしくなる不具合があります。
